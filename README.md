# Piyush Pradhan Portfolio

Modern React + Tailwind CSS portfolio website with Framer Motion animations, glassmorphism UI, resume download, profile photo, responsive navigation, contact validation, and production build support.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize

Update portfolio content in `src/data/portfolio.js`. Replace `public/profile.jpg`, `public/resume.pdf`, and files under `public/projects/` or `public/certificates/` when real assets are available.
